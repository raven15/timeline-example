TimeLine example
================

## 创建过程

### step.1 创建项目

- 创建基本目录结构

``` bash
mkdir -p timeline-example/{libs/{jquery,masonry},src/{css,js,images},example}
```

项目目录树如下:

```
.
└── timeline-example
    ├── libs
    │   ├── jquery
    │   └── masonry
    └── src
        ├── css
        ├── images
        └── js
cd timeline-example
```

- 使用 WebStorm 打开项目文件夹，创建 `.gitignore` 与 `README.md` 文件
